# Nimbus AI Dashboard - 4-Minute Demo Script

## Overview
This script provides a tight, compelling 4-minute demonstration of the Nimbus AI dashboard, highlighting key features while reinforcing the value proposition throughout.

**Total Time: 4 minutes**
- Main Dashboard: 90 seconds
- Reviews Page & Detail View: 90 seconds  
- AI Chat: 60 seconds
- Closing: 30 seconds

## üé® PRESENTATION TIPS

### **Pacing**
- Speak clearly and confidently
- Don't rush through the KPIs - they set the stage
- Pause briefly after showing AI-generated results (let them sink in)
- Leave 5-10 seconds for questions at the end if time permits

### **Highlighting**
- Use your mouse cursor to point at specific elements
- Hover over buttons before clicking to draw attention
- Read key numbers aloud (e.g., "70% high churn risk")
- Use phrases like "Notice here..." "What's powerful about this is..."

### **Energy & Enthusiasm**
- Show excitement about the AI features
- Use phrases like "This is where it gets really interesting..."
- Emphasize time savings and business impact
- Connect features to real business problems (churn, revenue, satisfaction)

### **Handling Technical Issues**
- If AI Summary or Chat is slow, say: "Claude is analyzing the review in real-time..."
- If something doesn't load, have a backup plan (screenshot or move to next feature)
- Stay calm and confident - the tech is impressive when it works
